# pokecli

## Install

```bash
$ npm install --global pokecli
```

## CLI

```bash
$ pokecli --help

  Usage
      $ pokecli

    Options
        --name The name of the pokemon 

    Examples
      $ pokecli --name=charmander
        Charmander
        ----------
        Metrics: 0.6m, 8.5 kg
        Type: fire

        Stats
        hp: 39
        attack: 52
        defense: 43
        special-attack: 60
        special-defense: 50
        speed: 65
```
